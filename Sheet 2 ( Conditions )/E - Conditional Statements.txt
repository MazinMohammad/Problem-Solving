Given a positive integer , do the following:

If , print the lowercase English word corresponding to the number (e.g., one for , two for , etc.).
If , print Greater than 9.
Input Format

A single integer, .

Constraints

Output Format

If , then print the lowercase English word corresponding to the number (e.g., one for , two for , etc.); otherwise, print Greater than 9.

Sample Input 0

5
Sample Output 0

five
Explanation 0

five is the English word for the number .

Sample Input 1

8

eight

Sample Input 2

44

Greater than 9

Solution:

#include <iostream>
using namespace std;

int main()
{
   int n;
   cin >> n;

   if (n == 1)
      cout << "one";
   else if (n==2)
      cout << "two";
   else if (n==3)
      cout << "three";
   else if (n==4)
      cout << "four";
   else if (n==5)
      cout << "five";
   else if (n==6)
      cout << "six";
   else if (n==7)
      cout << "seven";
   else if (n==8)
      cout << "eight";
   else if (n==9)
      cout << "nine";
   else
   cout << "Greater than 9";
}
