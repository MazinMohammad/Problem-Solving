B - Multiplication 1

Problem Statement:
Compute A×B


Constraints:
1≤A≤100
1≤B≤100
All values in input are integers.

Input:
Input is given from Standard Input in the following format:
A B

Output:
Print the value A×B as an integer.

Sample 1
Input	Output
2 5		10
We have 2×5=10.

Sample 2
Input		Output
100 100	10000


Solution:
#include <iostream>
using namespace std;

int main()
{
    int A, B;
    cin >> A >> B;
    cout << A*B;
}
